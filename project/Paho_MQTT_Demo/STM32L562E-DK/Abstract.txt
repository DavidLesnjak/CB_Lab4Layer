Paho MQTT Demo for STM32L562E-DK Board with WiFi Module

This demo application connects to Eclipse test server through MQTT,
subscribes to topics and publishes messages.

1. Configure WiFi connection settings in socket_startup.c:
 - SSID:          network identifier
 - PASSWORD:      network password
 - SECURITY_TYPE: network security

2. Select target:
 - STM32L562E-DK ESP8266:  ESP8266 WiFi Module (ARDUINO - USART)
 - STM32L562E-DK WizFi360: WizFi360 WiFi Module (ARDUINO - USART)
 - STM32L562E-DK ISM43362: ISM43362 WiFi Module (ARDUINO - SPI)

3. Build the application

4. Use default jumpers settings except set JP5 off (ARDUINO - SPI)

5. Connect the on board ST-Link Debugger via USB

6. Run the application and observe the messages in the Debug (printf) window
