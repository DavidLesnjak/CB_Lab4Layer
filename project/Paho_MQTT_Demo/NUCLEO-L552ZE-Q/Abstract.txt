Paho MQTT Demo for NUCLEO-L552ZE-Q Board with WiFi Module

This demo application connects to Eclipse test server through MQTT,
subscribes to topics and publishes messages.

1. Configure WiFi connection settings in socket_startup.c:
 - SSID:          network identifier
 - PASSWORD:      network password
 - SECURITY_TYPE: network security

2. Select target:
 - NUCLEO-L552ZE-Q ESP8266:  ESP8266 WiFi Module
 - NUCLEO-L552ZE-Q WizFi360: WizFi360 WiFi Module
 - NUCLEO-L552ZE-Q ISM43362: ISM43362 WiFi Module

3. Build the application

4. Connect the on board ST-Link Debugger via USB

5. Run the application and observe the messages in the Debug (printf) window
