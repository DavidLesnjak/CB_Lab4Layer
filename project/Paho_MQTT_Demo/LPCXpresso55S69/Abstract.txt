Paho MQTT Demo for LPCXpresso55S69 Development Board with WiFi Module

This demo application connects to Eclipse test server through MQTT,
subscribes to topics and publishes messages.

1. Configure WiFi connection settings in socket_startup.c:
 - SSID:          network identifier
 - PASSWORD:      network password
 - SECURITY_TYPE: network security

2. Select target:
 - LPCXpresso55S69 ESP8266:  ESP8266 WiFi Module
 - LPCXpresso55S69 WizFi360: WizFi360 WiFi Module

3. Build the application

4. Connect the on board DAP-Link Debugger via USB

5. Open a Terminal on PC and select the COM port of the DAP-Link Debugger
   Configure baudrate = 115200

6. Run the application and observe the messages in the Terminal
