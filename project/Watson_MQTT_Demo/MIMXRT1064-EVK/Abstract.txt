Watson MQTT Demo for MIMXRT1064-EVK Board with WiFi Module

This demo application connects to Watson IoT through MQTT
and publishes messages.

It requires a registered device in the Watson IoT Platform (IBM Cloud console).

1. Configure Watson IoT connection settings in helloWorld.c:
 - ORG_ID:      Organization ID
 - DEVICE_TYPE: Device Type
 - DEVICE_ID:   Device ID
 - TOKEN:       Authentication Token

2. Provide Server Certificate: IoTFoundation.pem on File System Drive M0:

3. Configure WiFi connection settings in socket_startup.c:
 - SSID:          network identifier
 - PASSWORD:      network password
 - SECURITY_TYPE: network security

4. Select target:
 - MIMXRT1064-EVK ESP8266:  ESP8266 WiFi Module
 - MIMXRT1064-EVK WizFi360: WizFi360 WiFi Module

5. Build the application

6. Connect the on board DAP-Link Debugger via USB

7. Open a Terminal on PC and select the COM port of the DAP-Link Debugger
   Configure baudrate = 115200

8. Run the application and observe the messages in the Terminal
